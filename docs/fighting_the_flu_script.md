## 0. 台本の前提

本台本は、ゲーム本編にストーリー演出を追加するための実装入力資料である。  
既存実装に、発言主表示・ステージ間シーン・BGM/SE連動・カットイン等が未実装でも、必要に応じて追加する前提で記述する。

ボム兵装は廃止済み。物語・セリフ・演出・UI文言ではボム兵装を使用しない。

## 1. 主要コンセプト

### 1.1 物語の一文要約

インフルエンザで高熱にうなされた澤口修平が、夢の中でウイルス、ネットミーム、婚活要塞、藤井竜王、そして自分自身の自己否定と戦い、相棒カロナール先輩の助けを得て「まだ投了しない」と決める話。

### 1.2 テーマ

- インフルそのものよりも、体調不良時に増幅される自己否定と戦う。
- カロナール先輩は人生を解決しない。だが、もう一手指す時間を作ってくれる。
- ラスボスを倒すのはカロナール先輩ではなく、澤口自身。
- エンディングでは現実の問題は解決していない。ただし、澤口は「投了しない」状態に戻っている。

### 1.3 カロナール先輩の役割

カロナール先輩は、序盤からプレイヤー機体の横に並ぶ随伴キャラ。  
態度はでかいが、通常時は弱い。

- 通常時: 弱い支援機。低威力ショット。被弾すると撤退。
- 性格: ふざけている。偉そう。だが性根はいいやつ。
- 退場: Stage 3後、承認欲求ブラックホールから澤口を作用反作用で押し出し、自分が落ちる。
- 復帰: ラスボス戦終盤、薬効発現により「カロナール先輩・薬効最大」として復帰。
- 復帰後: 強い。頼もしい。ラスボスの自己否定支配を弱める。

### 1.4 ラスボスの理屈

ラスボス「投了王サワグチ」は、澤口の自己否定スキーマが人格化した存在。  
高熱による認知ノイズの中で、自己否定が夢空間の生成ルールそのものになっている。

そのため、通常状態では以下の能力を持つ。

- 盤面改竄: フィールド、安全地帯、弾道を書き換える。
- 反芻再生: 後悔・弱音・被弾をエネルギーにして回復する。
- 投了勧告: 澤口の内側から発生する回避不能攻撃。
- 無限復帰: 澤口が自己否定を自分自身だと思い込む限り、HP0でも再生する。

カロナール先輩復帰後は、薬効により夢空間の認知ノイズが下がる。  
澤口は「自分の声」と「自己否定の声」を区別できるようになり、投了王の支配が崩れる。

```

## 3. キャラクター定義

### 3.1 澤口修平

31歳童貞。物理学修士。将棋有段者。  
自分なりに積み上げてきた自負がある一方で、婚活・仕事・私生活の停滞感に苦しんでいる。

特徴:

- 物性系出身だが、理論・量子情報・ブラックホール・宇宙ネタが好き。
- プライドが高いが、根は繊細。
- 自虐で先回りして傷つかないようにする癖がある。
- 最終的に「負ける可能性があること」と「今投了すること」を分けて考えられるようになる。

### 3.2 カロナール先輩

薬の精、あるいは高熱の夢の中に現れた解熱鎮痛薬的存在。  
存在自体はふざけているが、性根はかなり良い。

特徴:

- 自称「発熱界の潤滑油」。
- 口癖: 「熱は下げる。人生の難易度は下げられない」
- 通常時は弱い。
- 偉そうだが、澤口を否定しない。
- 本質は「根本解決ではなく、一手猶予を作る存在」。

通常時の戦闘性能案:

- 低威力ショット「微解熱弾」
- 低HP
- 被弾で撤退
- 一定時間後またはステージクリア後に復帰
- 撤退時セリフをランダム表示

復帰後の戦闘性能案:

- 名称: カロナール先輩・薬効最大
- 高威力ショット「解熱貫通弾」
- 敵弾相殺「鎮痛バリア」
- ラスボス回復停止「抗反芻フィールド」
- 回避不能攻撃対策「一手猶予バリア」

  "wait_for_input": true,
  "bgm": null,
  "se": "SE_TYPE",
  "fx": []
}
```

## 9.2 カロナール先輩随伴

通常時:

```text
- player の移動軌跡を追って後方から追従（左移動なら右から、右移動なら左から付いてくる）。
- 低威力ショットを自動発射。
- HPは1。被弾すると撤退し、被弾した敵弾は相殺、接触した敵にも反撃ダメージを与える。
- 被弾SE / 退場SE あり（dummy）。
- HP0（被弾）で一定時間（約24秒）撤退してから復帰。
- ステージ中の先輩機横セリフ（チャット）は廃止。
- Stage3後のイベントで完全離脱。
```

薬効最大時:

```text
- 右斜め後ろまたは上下に浮遊。
- 高威力の白色弾を発射。
- 投了王戦でのみ専用補助を有効化。
- 反芻再生停止フラグをON。
- 一手猶予バリアを一度だけ発動。
```

## 9.3 ラスボス能力の実装案

```text
盤面改竄:
- 一定時間ごとに安全地帯を移動。
- 弾の軌道を途中で曲げる。
- 画面端に「詰み」壁を出す。

反芻再生:
- カロナール先輩・薬効最大復帰前のみ有効。
- 5秒ごとに一定量回復。
- プレイヤー被弾時にも少量回復。

投了勧告:
- イベント攻撃。
- 復帰前は避けられず、HP1または瀕死まで削る。
- 復帰時に一手猶予バリアで無効化。
```

# 10. 重要な演出原則

1. カロナール先輩は最初から横にいる。弱いが、いないと寂しい存在にする。
2. カロナール先輩の退場は、友情ではなく、作用反作用・運動量保存・情報保存を混ぜたバカSFとして描く。
3. Stage 4前半は、カロナール先輩不在による静けさを出す。
4. ラスボスは「自己否定そのもの」なので、空間を支配できる。
5. カロナール先輩復帰後は、薬効で認知ノイズが下がり、自己否定の声と自分の声を分離できる。
6. カロナール先輩はラスボスを倒さない。最後の一撃は澤口が撃つ。
7. エンディングでは現実の問題を解決しない。投了しない状態に戻るだけでよい。
8. エンドロール後に、薬の用法・用量注意を必ず出す。
